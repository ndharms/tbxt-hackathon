Generated 2026-05-09 07:17

Approach

For each pair (A, B) of molecules within a fold, a binary classifier predicts whether molecule A is more potent than molecule B by ≥ 1.0 pKD unit. Features: [f(A) - f(B)] where f() = MACCS + RDKit descriptors + hashed Gobbi (640 features per pair).

Pairs are generated within each fold separately (no cross-fold pairs), preserving the Butina scaffold split. Training uses pairs from 4 folds; evaluation uses pairs from the held-out fold. Pair predictions are aggregated into per-molecule ranking scores, then compared to true pKD via correlation metrics.
